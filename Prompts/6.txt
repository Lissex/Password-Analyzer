Create frontend (HTML + CSS + vanilla JS) using Bootstrap 5.3 from CDN.
No bundler, no npm. Three files: index.html, style.css, app.js.

Load via CDN in index.html:
  Bootstrap CSS: https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css
  Bootstrap JS:  https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js
  JetBrains Mono font from Google Fonts

Override Bootstrap with custom CSS variables in style.css:
  --bs-body-bg: #0f0f13
  --bs-body-color: #e2e8f0
  --bs-border-color: #2d2d3d
  --bs-font-monospace: 'JetBrains Mono', monospace
  accent: #8b5cf6 (purple)

Layout — container-fluid, two columns (col-lg-6) on desktop, stacked on mobile:

LEFT col — Analyzer card (card bg-dark border-secondary):
  - card-header: "Password Analyzer" + Bootstrap badge со strength level
  - card-body:
      input-group: password <input class="form-control font-monospace">
                   + button "Show/Hide" (toggles type=password/text)
      mt-3: huge entropy display — custom div, not Bootstrap,
            animated counter via rAF, e.g. "73.2 bits"
      mt-3: strength bar — progress Bootstrap component, 5 steps,
            bg-danger / bg-warning / bg-info / bg-success / bg-purple
            (add .bg-purple in style.css: background: #8b5cf6)
      mt-3: character pool — 4 Bootstrap badges (badge rounded-pill):
            [abc] [ABC] [123] [!@#]
            inactive: bg-secondary, active: bg-purple text-white
      mt-3: crack times — Bootstrap list-group list-group-flush,
            4 items, each: scenario name (text-muted small) + time value right-aligned
            time color: text-success (>100yr) / text-warning (>1yr) /
                        text-danger-emphasis (>1day) / text-danger (instant)

RIGHT col — Generator card (card bg-dark border-secondary):
  - card-header: "Password Generator"
  - card-body:
      length slider: <input type="range" class="form-range"> 8–128
                     label shows live value "Length: 32"
      toggles row: 3x form-check form-switch in a row (gap-3 d-flex):
                   Uppercase / Digits / Symbols
      mt-3: btn btn-lg w-100 style="background:#8b5cf6" "Generate"
      mt-3: generated password display —
            input-group: <input class="form-control form-control-lg font-monospace" readonly>
                         + "Copy" button (btn-outline-secondary)
                         copy success: button turns btn-success + checkmark for 2s

Bottom row — History (full width, mt-4):
  card bg-dark border-secondary:
    card-header: "Recent analyses"
    card-body: Bootstrap table table-dark table-hover table-sm
      columns: Time | Entropy | Pool size | Strength
      last 5 from GET /api/v1/history
      fetch on page load + after each analyze/generate

Behavior (app.js):
  - Debounce analyze input 300ms (plain setTimeout/clearTimeout, no libs)
  - POST /api/v1/analyze on input, POST /api/v1/generate on button click
  - All DOM updates via plain JS — no jQuery
  - Copy: navigator.clipboard.writeText(), fallback execCommand
  - History: fetch on load, refresh after each action
  - Show Bootstrap spinner (spinner-border spinner-border-sm)
    inside Generate button while request is in flight
